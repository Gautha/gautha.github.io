Low cost hybrid GSM, Cloud based Remote Relay Control and Management System with Android application

Built a remote relay control to change the state of a relay using GSM band. The whole project was built using a ATtiny85 microcontroller and GSM modules. The device works by parsing an SMS and thus changing the state of any device connected to the relay. An android application has been built with cloud integration for multi-user operation of the relay. The back end database is powered by Parse. This solution has been developed for Panickker Switchgear as a cheap alternative for existing technologies. The whole device has been developed for less than 30$.
